# Visual Direction

Use this file when you need more concrete aesthetic guidance.

## Visual stance

Aim for premium clinical, not startup-generic and not spa-cliche. The interface should feel precise, composed, and human.

## Color system

- Base: warm white or cool porcelain backgrounds
- Structure: fog, stone, slate, or muted mineral neutrals
- Accent: one restrained medical or botanical accent such as deep teal, eucalyptus, ink blue, or muted copper
- Alert and success colors should be soft but readable, not saturated and game-like

## Typography

- Pair a characterful headline face with a highly legible body face.
- Good direction: elegant serif or humanist sans for headlines, neutral grotesk or practical sans for UI/body.
- Keep type scale generous. Let whitespace carry luxury rather than oversized effects.

## Layout

- Use asymmetry carefully to create premium rhythm.
- Alternate open sections with denser proof or data sections.
- Prefer 8px-based spacing systems with noticeably larger section spacing.
- Use 1 to 2 rounded corner radii across the system rather than many.

## Components

- Buttons: solid primary, quiet secondary, strong disabled contrast
- Cards: elevated through spacing, border tone, and shadow restraint
- Badges: factual labels, not decorative clutter
- Metrics: use only when they support decision-making or reassurance
- Testimonials: editorial layout, portrait optional, context always useful

## Motion

- Use motion to support clarity: reveal sequence, state transition, progress update
- Keep animation durations moderate and easing calm
- Avoid flashy hover effects and busy parallax

## Imagery

- Prefer real practitioners, real spaces, real product context
- Avoid generic smiling stock imagery and abstract anatomy renders
- If using illustration, keep it sparse and diagrammatic
